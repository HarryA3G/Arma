# Arma 3 Survivable Crashes Mod

**Author:** Rafael09ED

**Version:** 1.0

Allows the crew of a vehicle to survive it's destruction and escape harmed. 

Works with ACE and Vanilla. Activated and customizable with a module in the editor.
